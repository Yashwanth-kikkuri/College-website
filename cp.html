Syllabus

Module - 1: Basics

[by September 05, 2017] (For Mid Semester – 1)
1.Computers, Languages
2.Basic Structure of a C Program, compilation, execution
3.Data Types, Operators
4.Variables, constants, Expressions
5.Floating point arithmetic
6.Operators: Arithmetic, relational, conditional
7.Bitwise Operators
8.Input / Output functions
9.Conditional statements: ifelse, nestedifelse
10.Loops: for loop, while loop, dowhile
11.Switch case: break, continue


Module - 2: Intermediate Module

[by October 12, 2017] (For Mid Semester – 2)
1.1-Dimensional arrays
2.2-Dimensional Arrays
3.Strings, String operations
4.Functions: Built-in
5.Functions: User defined Functions
6.Functions with return type
7.Functions with strings
8.Functions with 2D arrays
9.Recursion: Simple and advanced


Module - 3: Advanced Module

[by November 10, 2017] (For End Semester)
1.Pointers: Call by value
2.Pointers: Call by address
3.Structures: Simple and Array of structures
4.Functions with structures
5.Dynamic Memory Allocation: List
6.Dynamic Memory Allocation: Queue, Stack
7.File Handling: File Operations
8.Preprocessor directives
9.Command line arguments
10.User defined header files
11.A few more “Practical Problems”
